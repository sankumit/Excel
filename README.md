# Web Utility for Excel File Management

This repository contains a web utility for handling Excel files with features for file uploads, analysis, and data visualization.

## Directory Structure
```
/excel
  ├── /frontend
  │   ├── index.html
  │   ├── style.css
  │   └── app.js
  ├── /backend
  │   ├── server.js
  │   └── fileUpload.js
  └── README.md
```  

## Setup Instructions

### Frontend
1. Navigate to the `/frontend` directory.
2. Open `index.html` in a web browser to view the dashboard.
3. Modify `style.css` for custom styling and `app.js` for JavaScript functionality.

### Backend
1. Navigate to the `/backend` directory.
2. Run `npm install express multer` to install necessary dependencies.
3. Use `node server.js` to start the server for handling file uploads and data processing.
